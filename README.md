## Descrição
Este projeto é uma aplicação web simples e intuitiva criada para facilitar o estudo e o teste de filtros digitais aplicados a imagens.

## Como Contribuir (Adicionar Novos Filtros)
- Fork este repositório para sua conta.

- Crie uma branch de desenvolvimento (seguindo o padrão DOCB{id}).

- Implemente seu novo filtro e teste-o.

- Faça o commit das suas alterações (seguindo o padrão Conventional Commits).

- Abra um Pull Request (PR) para que suas mudanças sejam revisadas e integradas.
